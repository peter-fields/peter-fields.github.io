# Instructions: Phase Diagram of a Driven Izhikevich Neuron

---

## The Model

### 1. Izhikevich Neuron

The neuron has two variables: membrane voltage v and a recovery variable u.

```
dv/dt = 0.04*v^2 + 5*v + 140 - u + A*x(t)
du/dt = a*(b*v - u)
```

When v reaches 30 mV, a spike is recorded and the variables are reset:
```
if v >= 30:
    record spike time
    v = c
    u = u + d
```

v is in mV, t is in ms. A is the input strength (one of your two phase diagram axes).

### 2. Stochastic Damped Harmonic Oscillator (SDHO)

The input signal x(t) is generated by a stochastic damped harmonic oscillator. Write it as two first-order ODEs:

```
dx/dt = y
dy/dt = -2*gamma*omega_0*y - omega_0^2*x + sigma*xi(t)
```

where xi(t) is Gaussian white noise (see below). sigma is the noise amplitude (your second phase diagram axis). Suggested fixed parameters: omega_0 = 2*pi*10 (10 Hz oscillator), gamma = 0.1 (lightly damped). Initial conditions: x(0) = 0, y(0) = 0.

### 3. Noise

xi(t) is Gaussian white noise. In discrete Euler integration at timestep dt, sample it as:

```
xi = randn() / sqrt(dt)
```

so that the noise term contributes sigma * randn() * sqrt(dt) to y at each step. This is the standard Ito convention for white noise in discrete time.

---

## Simulation

Use Euler integration with dt = 0.1 ms. Simulate for T = 3000 ms total. Discard the first 500 ms as transient. Collect spike times from the remaining 2500 ms.

At each timestep:

```
x  +=  y * dt

y  +=  (-2*gamma*omega_0*y - omega_0^2*x) * dt  +  sigma * sqrt(dt) * randn()

dv  =  0.04*v^2 + 5*v + 140 - u + A*x

du  =  a*(b*v - u)

v  +=  dv * dt

u  +=  du * dt

if v >= 30:
    record spike time
    v = c
    u = u + d
```

## Observables

### Vector Strength R

R measures how consistently the neuron fires at the same phase of the input oscillation. T_input = 1/f_0 = 100 ms is the period of the oscillator.

For each spike time t_k, compute its phase within the input cycle:

```
phi_k = 2*pi * (t_k mod T_input) / T_input
```

Then:

```
R = | (1/N) * sum_k exp(i * phi_k) |
```

where N is the total number of spikes and i is the imaginary unit. R = 1 means every spike falls at exactly the same phase (perfect locking). R = 0 means spikes are uniformly distributed across all phases (no locking).

If N = 0 (no spikes), set R = NaN.

### Coefficient of Variation of ISI (CV)

The interspike interval (ISI) is the time between consecutive spikes. If spike times are t_1, t_2, ..., t_N then ISI_k = t_{k+1} - t_k.

```
CV = std(ISI) / mean(ISI)
```

CV = 0 means perfectly regular firing (spikes are evenly spaced). CV = 1 means Poisson-like irregular firing. CV > 1 means highly irregular or bursty firing.

If N < 2 (fewer than 2 spikes), set CV = NaN.

## Plots

Make two side-by-side imshow plots on the same (A, sigma) grid:

1. **R** — use a colormap from 0 (white or light) to 1 (dark/saturated). NaN cells (silent) should be a distinct color (e.g. grey).
2. **CV** — use a colormap from 0 to ~2. NaN cells in grey.

Label the x-axis A (input strength) and the y-axis sigma (noise). Add a colorbar to each.

You should expect to see roughly four regions:
- **Silent** (R = NaN, CV = NaN): low A, any sigma
- **Synchronous** (R high, CV low): high A, low sigma
- **Asynchronous** (R low, CV moderate): high A, moderate sigma
- **Disordered** (R low, CV high): high sigma

## Appendix: What Do a, b, c, d Mean?

The Izhikevich neuron has four parameters that control the neuron's intrinsic firing behavior.

- **a** controls how fast the recovery variable u relaxes back to its resting value. Small a means slow recovery — the neuron stays suppressed longer after each spike and fires less readily.
- **b** controls how strongly u is coupled to v at rest. Larger b makes the neuron more resonant near threshold.
- **c** is the voltage reset after each spike. More negative c means the neuron has to climb further back up to threshold before it can fire again.
- **d** is how much u jumps up at each spike. Larger d means stronger after-spike adaptation — the neuron slows down after firing in a burst.

The values a=0.02, b=0.2, c=-65, d=8 define a **regular spiking** neuron — the most common type of excitatory cortical cell. It fires tonically in response to sustained input and shows mild rate adaptation (gradual slowdown) under strong drive. This is the standard starting point in the Izhikevich model literature.
